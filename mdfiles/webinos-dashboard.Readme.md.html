webinos-dashboard ================= Dashboard API of webinos ### Explorer Module ### Module: "explorer" #### Data #### - *service*
String or array of strings. The services you want to filter/discover.
Default: "*" - *select*
String. The type you want to be selected.
Allowed: "people", "devices", "services", "none"
Default: "services" - *show*
String. The type you want to show.
Allowed: "people", "devices", "services"
Default: "services" - *multiselect*
Boolean. If you want to allow multiple selections.
Default: false #### Result #### Result is always an array of objects containing the selections. The properties of the objects depends on the user selected types. - *id* - *type* - *friendlyName* - *api*
(only if the type is service) - *address*
(only if the type is service) - *deviceType*
(only if the type is device) ### ServiceModule ### Module: "serviceConfiguration" #### Data #### - *params*
Object containing information to configure a spefic API or null (see "params" element in config.json). A page to configure APIs (the one selected or all of them) will be shown in the dashboard. Default: "null" #### Result #### No result returned.
